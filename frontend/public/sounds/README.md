# Sons du projet OnlyRoll

## Konami Code Sound

Pour le son du Konami Code, vous devez ajouter un fichier audio nommé `konami.mp3` dans ce dossier.

### Options recommandées :

1. **Télécharger un son 8-bit gratuit** :
   - [Freesound.org](https://freesound.org/search/?q=8bit+victory) - Recherchez "8bit victory" ou "retro game win"
   - [Zapsplat.com](https://www.zapsplat.com/) - Section "Video Game Sounds"
   - [Mixkit.co](https://mixkit.co/free-sound-effects/video-game/) - Sons de jeux vidéo gratuits

2. **Générer votre propre son** :
   - [Bfxr.net](https://www.bfxr.net/) - Générateur de sons rétro en ligne
   - [Chiptone](https://sfbgames.itch.io/chiptone) - Outil de création de musique chiptune

3. **Utiliser un extrait Konami classique** :
   - Recherchez "konami victory sound" sur YouTube et utilisez un convertisseur MP3

### Format requis :
- **Nom du fichier** : `konami.mp3`
- **Format** : MP3 (recommandé pour la compatibilité navigateur)
- **Durée** : 1-3 secondes idéalement
- **Taille** : < 100 KB

### Alternative sans son :
Le composant fonctionne également sans fichier audio. Si le fichier est absent, l'animation et les confettis s'afficheront quand même sans erreur.
